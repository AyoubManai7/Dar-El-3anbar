### Installation

1. Clone the repository
```bash
git clone https://github.com/AyoubManai7/Dar-El-3anbar.git
cd Dar-El-3anbar
```

2. Set up the database
```sql
CREATE DATABASE dar_el_3anbar;
```

3. Configure database connection in `config/db.php`

4. Install frontend dependencies
```bash
npm install
npm run dev
```
